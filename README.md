# 🩺 Chronic Kidney Disease (CKD) Prediction AI Agent

## About the Project

The Chronic Kidney Disease (CKD) Prediction AI Agent is a machine learning-based healthcare application that helps predict the risk of chronic kidney disease using patient health information. The system analyzes medical parameters, provides an instant prediction, generates a health report, and offers basic health recommendations.

The project is built using **Python**, **Scikit-learn**, and **Streamlit** to create a simple and interactive web application. Users can enter patient details through the interface, and the trained machine learning model predicts whether the patient is at risk of CKD.

### Features

* 🩺 Predicts Chronic Kidney Disease risk
* 📊 User-friendly Streamlit web interface
* 🤖 Machine learning-based prediction
* 📄 Generates patient health reports
* 💡 Provides basic health recommendations
* ⚡ Fast and easy to use

### Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Joblib

### How It Works

1. Enter the patient's medical details.
2. Click the **Predict** button.
3. The AI model analyzes the data.
4. The application displays the prediction result.
5. A health report and recommendations are generated.

> **Note:** This project is intended for educational and research purposes only. It is not a substitute for professional medical diagnosis or treatment.
